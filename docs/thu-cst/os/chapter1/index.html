# 操作系统抽象

## CPU

- 进程

## 磁盘

- 文件

## 内存

# 操作系统内核的特征

## 并发

- 计算机系统中同时**存在**多个运行程序

## 共享

- 程序间“同时”访问**互斥**共享各种资源

## 虚拟

- 每个程序” 独占” 一台完整的计算机

## 异步

- 服务的完成时间不确定，也可能失败
