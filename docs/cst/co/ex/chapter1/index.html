计算机硬件

习题1

冯.诺依曼机的基本思想是 (C) 。
A.用二进制表示信息
B.以控制器为核心
C.存储程序控制
D.用十六进制表示指令

习题2

以下有关冯.诺依曼机的叙述中，错误的是 (D)。
A.计算机硬件由运算器、控制器、存储器、输入设备/输出设备5大部件组成
B.指令由操作码和地址码两部分组成
C.机器以运算器为中心，输入/输出设备与存储器间的数据传送通过运算器完成
D.操作数直接包含在指令中

解析：指令由操作码和地址码组成:
● 操作码用来表示执行何种操作
● 地址码用来表示操作数在存储器中的位置

习题3

以下有关计算机各功能部件的叙述中，错误的是 (D)。
A.存储器的主要功能是存储程序和数据
B.控制器的主要功能是解释存储器中的指令，并发出各种操作命令来执行指令，I/O设备也受控于控制器
C.输入设备/输出设备的主要功能是完成用户与计算机之间的信息交互
D.运算器的主要功能是进行算术运算

解析：运算器的核心是算术逻辑单元ALU,主要功能如下:
● 算术运算:加、减、乘、除
● 逻辑运算:与、或、非、异或、移位等
