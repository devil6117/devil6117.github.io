《算法导论》思考题 3-4

Inline ：$$(a+b)^2=a^2+2ab+b^2$$

Inline ：$(a+b)^2=a^2+2ab+b^2$

Inline ：$(a+b)^2=a^2+2ab+b^2$

问题
解析
$$(渐近记号的性质)\ 假设\ f(n)\ 和\ g(n)\ 为渐近正函数。\证明或反驳下面的每个猜测。$$
注：蕴涵，即可推出。例：A蕴涵B，即可由A推出B。
$$a.\ f(n)=O(g(n))\ 蕴涵\ g(n)=O(f(n))。$$
$$❌\ 反例 g(n)=n^2，f(n)=n。但是\ n^2\ 不等于\ O(n)。$$
$$大O记号表达的是一个渐近函数，而非具象函数。$$
$$多个具象函数可以对应同一个渐近函数，但特定某个渐近函数是无法反推出原具象函数。$$
$$b.\ f(n)+g(n)=Θ(min(f(n)，g(n)))$$
$$❌\ n^2+n\neq Θ(min(n^2,n))=Θ(n)，$$
$$Θ必须取最高阶中的下界，即Θ需严格同阶。$$
$$c.\ f(n)=O(g(n))\ 蕴涵\ lg(f(n))=O(lg(g(n)))，$$
$$其中对所有足够的n，有 lg(g(n))\geq 1且f(n)>1。$$
$$✅\ 正数取对数不改变大小关系。
证明:\exist c,n_0>0,\ \forall n\geq n_0\\ \ \ \ \ \ \ \ \ \ \ \ f(n)\leq c\cdot g(n),lg(f(n))\leq d\cdot lg(g(n))\\ \ \ \ \ \ \ \ \ \ \ \ d=\frac {lg\ c+lg\ g(n)}{lg\ (n)}=\frac {lg\ c}{lg\ g(n)}+1\leq lg\ c+1$$
$$d.\ f(n)=0(g(n))\ 蕴涵\ 2^{f(n)} =O(2^{g(n)})。$$
$$❌\ 可以取对数，但不能取指数。反例：f(n)=2n,g(n)=n。$$
$$O(f(n))=O(g(n)),但是2^{f(n)}=(2^2)^n=4^n,与\ 2^{g(n)}=O(2^n)\ 不相等。$$
$$f.\ f(n)=O(g(n))\ 蕴涵\ g(n)=2(f(n))。$$$$✅\ 证明：\exist c,n_0>0,\ \forall n\geq n_0$$
$$f(n)\leq c\cdot g(n),lg(f(n))\leq d\cdot lg(g(n))$$
$$d=\frac {1}{c},当n\geq n_0,g(n)\geq d\cdot f(n)，g(n)=\Omega (n)$$
$$g.\ f(n)=Θ(f(n/2))$$$$❌\ 反例：f(n)=2^{2n}\neq Θ(2^n)，而f(\frac {n}{2})=2^n$$
$$h.\ f(n)+o(f(n))=Θ(f(n))$$$$✅证明：\exist c,n_0\geq 0,\forall n\geq n_0,0\leq g(n)\leq c\cdot f(n)$$
$$exist c_1,c_2,n_0,\forall n\geq n_0,0\leq c_1\cdot f(n)+g(n)\leq c_2\cdot f(n)$$
