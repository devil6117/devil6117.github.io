VS Code 安装

2024-02-19
阅读时长: 1 分钟

系统环境

Windows 10x64

获取安装包

1、打开官网下载页面

https://code.visualstudio.com/Download

2、下载system-x64版

安装VS Code

(安装过程略)

配置中文语言(可选项)

注：个人建议全英文环境使用，有利于培养学术思维。

（1）主界面按下Ctrl+Shift+P

（2）输入Configure Display Language

（3）选中“中文(简体)”

（4）重启VS Code
